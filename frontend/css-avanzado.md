# CSS Avanzado

CSS permite diseñar y presentar documentos HTML de forma atractiva.

## Conceptos clave
*   **Flexbox:** Diseño de cajas flexibles.
*   **Grid:** Maquetación en cuadrícula 2D.
*   **Animaciones:** Transiciones y movimientos.
