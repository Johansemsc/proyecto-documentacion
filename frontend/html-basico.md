# HTML Básico

HTML es el lenguaje estándar para crear páginas web.

## Elementos principales
*   **Etiquetas:** Definen la estructura.
*   **Atributos:** Proporcionan información extra.
*   **Semántica:** Da significado al contenido.

## Referencia
Consulta la [Referencia HTML de MDN](https://developer.mozilla.org/es/docs/Web/HTML).
