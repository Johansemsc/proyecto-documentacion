# JavaScript Moderno (ES6+)

JavaScript ha evolucionado enormemente en los últimos años.

## Nuevas características
*   **Arrow Functions:** Sintaxis más corta para funciones.
*   **Promesas:** Mejor manejo de la asincronía.
*   **Módulos:** Organización del código en ficheros.
