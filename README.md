# Guía Completa de Desarrollo Web

Este repositorio contiene una documentación técnica detallada sobre el desarrollo web moderno.
Abarca desde los fundamentos de frontend y diseño hasta la arquitectura de backend y seguridad,
sirviendo como proyecto para demostrar el dominio del control de versiones con Git y GitHub.
