# Servidores Web

El servidor web es el software encargado de procesar las peticiones de los clientes.

## Ejemplos Populares
*   **Apache:** Servidor robusto y muy extendido.
*   **Nginx:** Especializado en alto rendimiento.
*   **IIS:** Servidor propietario de Microsoft.
