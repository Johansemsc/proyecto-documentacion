# APIs REST

REST es un estilo de arquitectura para diseñar servicios web.

## Métodos HTTP
Las operaciones se basan en verbos estándar:
*   **GET:** Obtener información.
*   **POST:** Crear nuevos recursos.
*   **PUT/DELETE:** Actualizar o borrar.
