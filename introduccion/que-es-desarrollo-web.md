# ¿Qué es el Desarrollo Web?

El desarrollo web es el proceso integral de construcción de sitios y aplicaciones para internet.

## Áreas principales
El desarrollo se divide típicamente en dos grandes áreas:

*   **Frontend:** Lo que el usuario ve e interactúa.
*   **Backend:** La lógica del servidor y bases de datos.
